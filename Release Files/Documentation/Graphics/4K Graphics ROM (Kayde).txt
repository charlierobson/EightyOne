============================
4K Graphics ROM for the ZX81
Kayde
============================

The Kayde 4K Graphics ROM is identical to the dk'tronics 4K Graphics ROM but
with the exception that it has a different set of definitions for character set 6.


CHARACTER SET No 6
ADDRESS FROM 10240 TO 10751 STEP 8
USR 11924
