============================
4K Graphics ROM for the ZX81
dk'tronics
============================

USER DEFINABLE GRAPHICS

Once you have fitted your 4K Graphic Rom, you may wish to have user
Definable Graphics.

This is accomplished very quickly and easily by fitting a RAM into the empty
holder.

1K of Ram will give you 128 programmable graphics, 2K of Ram gives 256. The
Ram Address starts at 12288 and it will be necessary to Poke to 8 locations
to create each new graphic.

Graphics may also be taken out of Eprom (Graphic ROM) and loaded into RAM
by the simple use of Peek and Poke.


USING YOUR GRAPHIC ROM

The following program will print your new Graphics 64 at a time:

  10 INPUT A
  20 CLS
  30 FOR X=0 TO 63
  40 PRINT CHR$(X);" ";
  5O NEXT X
  60 LET L=USR A
  70 GOTO 10

When you RUN the program it will be waiting for an INPUT. Enter one of the
following numbers:

11004 = Set 1: MISC.
11909 = Set 2: INVADERS & MISC.
11914 = Set 3; UPPER & lower CASE.
11919 = Set 4: MISC.
11924 = Set 5: TRUE INVERSE.
11929 = Set 6: MISC.
11934 = Set 7: MISC.
11939 = Set 8: SINCLAIR'S GRAPHICS
11944 = Set 1: U.D.G.
11949 = Set 2: U.D.G.

The important part of the program was the USR line : LET L=USR A

Now type NEW and enter the following as a Direct command: LET L=USR 11909

You will find that you now have different graphics when you press the keys.

Try it again with one of the other numbers, 11924, etc.

The number 11939 will bring you back to Sinclair's Graphics.

When writing programs, put in a line number 9999 LET L=USR 11939, because
when you have changed character sets it may not always be possible to read
your listings. Then all you have to do is to type in GOTO 9999 and this will
put you back into the Sinclair set.

Try the following:

  10 LET L=USR 11909
  20 PRINT AT 8,10;"A  B  C"; AT 12,10;"DE FG HI JK"
  30 PAUSE 200
  40 GOTO 9999
9999 LET L=USR 11939

Now try these other simple programs:

a)  10 LET L=USR 11939                 This will give you an exploding
    20 LET L=USR 11924                 screen effect; to stop all you
    30 GOTO 10                         need to do is press BREAK.

b)   5 LET L=USR 11904
    10 PRINT "AB AB AB"                Useful in a lot of games.

c)  10 LET L=USR 11929
    20 PRINT "I";
    30 GOTO 20                         Humpty Dumpty

d)  10 LET L=USR 11929
    20 PRINT "FHHHHHHHHHHHHHHHHHHHHHHHHHHHHHHG"
    30 FOR X=1 TO 19
    40 PRINT "A"; TAB 31;"B"
    50 NEXT X
    60 PRINT "CDDDDDDDDDDDDDDDDDDDDDDDDDDDDDDE"
    70 PRINT AT 8,10;"vx  vx"          (vx = Shifted VX)
    80 PRINT AT 16,12;"KL   KL"

The above programs do not mean very much on their own and are intended just
to give you an idea or two on how the GRAPHIC ROM may be used to advantage in
the construction of either simple or very elaborate games; several of which
are available direct from us on Cassette and employ machine Code snd Basic
to give very fast and addictable games, using the capabilities of the GRAPHIC
ROM to the full.


PRINTING GRAPHICS TO THE SINCLAIR PRINTER

If you are using a large amount of memory, you will have to lower RAM TOP.

RUN 9993 first then GOTO 9983 tor Graphic Copy.

This routine will copy to the printer the contents of the D/File when called.

9983 FAST
9984 LET S=PEEK 16396+PEEK 16397*256
9986 FOR L=0 T0 23
9986 FOR C=1 TO 33
9987 POKE 16443+C,PEEK(S+C)
9988 NEXT C
9989 LET P=USR 31744
9990 LET S=S+33
9991 NEXT L
9992 STOP
9993 FOR I=0 T0 112
9994 POKE 31744+I,PEEK (2161+I)
9995 NEXT I
9996 POKE 31800,?
9997 POKE 31857,201
9998 STOP
9999 LET L=USR 11939

NOTE: THE ? IN LINE 9996 MUST BE GIVEN ONE OF THE FOLLOWING Nos.

16 = GRAPHIC SET 11904
17 = GRAPHIC SET 11909
18 = GRAPHIC SET 11914
19 = GRAPHIC SET 11919
20 = GRAPHIC SET 11924
21 = GRAPHIC SET 11929
22 = GRAPHIC SET 11934
